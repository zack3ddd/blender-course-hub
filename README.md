# Blender 課程資源中心

給 zack3d Blender 課程學員的插件安裝與問題排查資源。

## 👉 直接開這裡使用

**https://zack3ddd.github.io/blender-course-hub/**

選好你的 Blender 版本（3.6 / 4.5 / 5.2）與作業系統，頁面會自動篩出對應的插件安裝方式、相容性、常見問題與硬體建議。

## 內容

- **插件查詢**：課程用到的插件，各版本相容性、官方頁面與下載連結、安裝步驟
- **問題排查**：常見狀況與解法
- **硬體建議**：依顯卡等級的渲染參數
- **更新日誌**：插件與版本異動

## 維護說明

所有資料集中在 `index.html` 開頭那段有註解框起來的資料區（`PLUGINS` / `FAQS` / `GPUS` / `LOGS`）。要新增或修改插件，只需要動那一段，push 之後 live 頁面就會自動更新。這裡是唯一的資料真相來源。
